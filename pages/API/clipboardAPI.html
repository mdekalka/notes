Clipboard API

Определение

Clipboard API дает способ приложению выполнять такие команды как cut/copy/paste, вызванные действиям июзера.

Обзор свойств

  • document.addEventListener('cut/copy/paste', eventHandler) - ивент,который триггеррится в зависимости от выполненного юзером действия.
  • event.clipboardData.setData('text/plain', data) - определяет данные, которые будут записаны в clipboard при cut/copy операциях.
  • event.clipboardData.getData('text/plain') - возвращает данные,которые были определены в clipboard через paste операцию.
  • document.execCommand('cut/copy/paste') - программно вызывает выбранную операцию на данных или элементе, который находится в фокусе.
