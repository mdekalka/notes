Fullscreen API

Определение

Fullscreen API позволяет приложению переходить в фулл-скрин режим.

Обзор свойств

  • document.fullscreenEnabled - возвращает boolean значение, показывающее доступен/разрешен ли фулл-скрин режим.
  • element.requestFullscreen() - отправляет запрос на фулл-скрин режим для выбранного элемента. Возвращает resolved Promise, если запрос был успешен.
  • document.fullscreenElement - возвращает ссылку на элемент, который сейчас находится в фулл-скрин режиме.
  • document.addEventListener('fullscreenchange', eventHandler) - ивент, который триггерится на включение/отключение фулл-скрин режима.

// Передать элемент и сделать запрос на фулл-скрин режим
function onFullScreenHandlerThis(element, event) {
    var _this = this;

    if (element) {
        var fsPromise = element[requestFullscreen]();

        if (fsPromise && fsPromise.catch) {
            maybePromise.catch(function(error) {
                _this.addLog('Cannot acquire fullscreen mode: ' + error);
            });
        }
    }
};

// Закрыть фулл-скрин режим
document.exitFullscreen();
